# Jogo da Memória

Este é um jogo da memória desenvolvido com **React**, **Vite** e **TypeScript**. O mesmo possui alguns recursos como:

- Animação de flip nas cartas com [Framer Motion](https://www.framer.com/motion/)
- Contador de jogadas e temporizador
- Efeito visual (confete) ao finalizar o jogo com [canvas-confetti](https://www.npmjs.com/package/canvas-confetti)

## Funcionalidades

- **Jogo interativo:** As cartas iniciam viradas para baixo e só são reveladas ao serem clicadas.
- **Verificação de pares:** Ao selecionar duas cartas, o jogo checa se elas formam um par. Se forem iguais, permanecem viradas. Caso contrário, são viradas de volta.
- **Contador de jogadas e temporizador:** Exibe o tempo decorrido e o número de jogadas realizadas.

## Tecnologias Utilizadas

- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- [Framer Motion](https://www.framer.com/motion/)
- [canvas-confetti](https://www.npmjs.com/package/canvas-confetti)
